Tahti VR (2021-2022)

A VR rhythm game where you shoot targets and dodge obstacles to the music, made with Unity.

This project was originally for a school assignment that is now finished, but it is still in active development and we plan on releasing it in the future. We are working in a team of four people.

My responsibility has been programming most of the core gameplay mechanics, MIDI-songmap importing, as well as making some of the music (e.g. the one you can hear on the gameplay video). For MIDI importing, I used a library called DryWetMIDI. The game allows anyone to make their own levels with custom music, by using any software capable of MIDI editing.
