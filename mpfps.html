Multiplayer FPS Prototype (2021)

A class-based FPS game prototype with working multiplayer I made using Unity during the autumn of 2021.

The goal of this project is to make strategic and fast-paced multiplayer shooter. The players are split into two teams with the size of 2-4. There would be around 5 classes with unique stats (armor, speed), and with two weapons and a utility item each (e.g. a grenade). In the primary gamemode, the objective would be to take a single flag from the middle of the map to the enemy base.

Notable features:

  • Client-server architecture using LiteNetLib
  • Functioning online multiplayer
  • Modular class system
  • Modular weapon system
  • One fully fledged class
  • Skill based movement system, including mechanics like crouchsliding, explosive jumps, aircontrol
