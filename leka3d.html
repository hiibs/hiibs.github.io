Leka3D (2021)

A game engine prototype I made using C++ and OpenGL during the summer of 2021.

Notable features:

  • Modular game objects that can form hierarchical structures
  • Easily import custom models and textures
  • Collision physics
  • Raycasting
  • Basic lighting (directional, ambient, multiple point lights)
  • Billboard particle effects
  • UI overlay images
  • Input system
  • Supports multiple camera layers (e.g. rendering the world and the weapon model separately)
