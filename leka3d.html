Leka3D (2021)

A 3D game engine I made using C++ and OpenGL during the summer of 2021.

Notable features:

  • Modular game objects that can form hierarchical structures
  • Easily import custom models and textures
  • Collision physics, AABB against polygon (still somewhat buggy)
  • Raycasting against polygons (also somewhat buggy)
  • Basic lighting (directional, ambient, multiple point lights)
  • Animated spritesheet particle effects
  • UI overlay images
  • Input system
  • Supports multiple camera layers (e.g. rendering the world and the weapon model separately)
